### Simple security roadmap

**Short-term roadmap:** survive till you get secure

* Quick security assessment
* Easy fixes for easy to identify threats


**Long-term roadmap:** get secure

* Identify risks
* Prioritise mitigations
* Model threats
* Set up controls 
* Deploy
* Monitor
* Iterate
